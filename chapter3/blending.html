Blended Point Spread Functions

Measuring Flux

Blending results in light from the neighbors contaminating the flux of the lensed star, fs, measured at time t.

For an isolated lensed star, the measured flux f(t) is given by:

$$f(t) = f_{s}(t)A(u(t))$$
In practise what we actually measure is:

$$f(t) = f_{s}(t)A(u(t)) + f_{b}(t)$$

where A(u(t)) is the magnification at angular separation u, and fb(t) is the additional flux from all overlapping neighbors.

This is strongly time-dependant: for instance, if the seeing suddenly worsens during observations, the stellar PSFs expand radially and fb abruptly increases.

Blending in microlensing lightcurves

Blended flux can "dilute" the measured amplitude, and make tE appear shorter, leading to a mis-estimation of the lens parameters.

As this can be a symmetrical change in the lightcurve, it can mimic the affects of the parallax component πE,⊥.

Notation

Different authors sometimes use different formulations for blending, so you may also see the blend ratio, g:
$$g = f_{b}/f_{s}$$

Dealing with Blending

Contemporaneous sets of observations from different telescopes and instruments can help here.

Each dataset has a different combination of telescope aperture, instrument resolution and seeing conditions, as in these examples, so fs and fb will vary independently for each dataset.

So we consider both as functions of time for each dataset, k (with Nk datapoints), in turn:

$$f_{k}(t) = f_{s,k}(t)A(u(t)) + f_{b,k}$$

Measuring the blend flux

A(u(t)) can usually* be considered to be the same for all datasets from Earth.

We derive a starting model A(u(t)) by adopting typical initial values for the physical event parameters, which we can use to calculate the measured flux, fm,k(t) (with measurement errors, Ïƒm,k(t)). We can test how well the model fits with a χ2 test:

$$\chi^{2} = \frac{\sum_{i} \left( \frac{f_{m,k}(t) - f_{k}(t)}{\sigma_{m,k}(t)} \right)^{2}}{N_{k} - N_{par}}$$

Npar is the number of fitted parameters.

* We'll come back to this later

Measuring the blend flux

Differentiating this expression with respect to each variable in turn and equating it to zero, we can derive a set of simultaneous equations which can be used to infer fs,k, fb,k for each dataset, for the given amplification model.

Given this set of initial blending parameters, we can now repeat this process, but this time fit for the physical parameters of the amplification model. By iterating this process, we can arrive a set of consistent parameters.

Stellar variability

So far we have assumed that the blended neighbors to the lensed star give constant flux over the duration of the lensing event. Of course, that isn't always the case, and if one or more neighboring stars has intrinsic variability, fb may also vary with time and wavelength.

This can be accommodated by substituting fb(t) with an appropriate function, for instance a linear gradient or sinusoid. Observations taken before or after the lensing event can be used to determine a suitable model.

Widely-spaced Observatories

We can only assume that the lensing magnification, A(u(t)), is the same for all observers if they share the same line of sight to the event.

If observers are widely separated (e.g. on Earth and a spacecraft), then the observers will measure different angular separations u at time t.

If this is the case, then we have to take this into account when computing A(u(t)).
